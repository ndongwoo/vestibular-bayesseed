# Highlights

- Vestibular-BayesSeed provides modular JSON definitions for logistic Bayesian diagnostic networks.
- The framework separates disease-specific clinical knowledge from reusable inference code.
- Default worked modules are included for BPPV, Ménière's disease, and presbyvestibulopathy/BVP.
- The package includes CLI execution, Streamlit demonstration, synthetic cases, schema validation, and tests.
- The software is designed for research and education, not for unvalidated clinical deployment.
