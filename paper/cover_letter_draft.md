# Cover letter draft

Dear Editors,

We submit **“Vestibular-BayesSeed: An Open-Source Framework for Evidence-Anchored Logistic Bayesian Networks in Dizziness Diagnosis”** as an Original Software Publication for *SoftwareX*.

Vestibular-BayesSeed is an open-source Python framework for constructing evidence-anchored logistic Bayesian diagnostic networks using modular JSON disease definitions. The software separates disease-specific clinical knowledge from reusable inference code, allowing users to define input nodes, derived nodes, target disease nodes, coefficients, evidence metadata, and sensitivity ranges without modifying the core inference engine.

The initial release includes worked example modules for benign paroxysmal positional vertigo, Ménière’s disease, and presbyvestibulopathy with bilateral vestibulopathy as a competing threshold-based diagnosis. The repository also includes schema validation, a command-line interface, synthetic demonstration cases, unit tests, documentation, and an optional Streamlit demonstration interface.

The software is intended for research and educational use as a seed framework for clinical knowledge engineering. It is not presented as a validated clinical decision-support system, medical device, or diagnostic tool for patient management. No patient-level data are included in the repository.

We believe this submission fits *SoftwareX* because the main contribution is reusable research software, not a conventional clinical validation study. The package is designed to be inspected, executed, tested, and extended by other researchers interested in transparent probabilistic diagnostic modeling.
