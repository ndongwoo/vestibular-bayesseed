# Table 3. Default worked disease modules

| Module | Diagnostic phenotype represented | Key input nodes | Key derived nodes | Demonstration role |
|---|---|---|---|---|
| BPPV | Positional and nystagmus-driven vestibular disorder | Positional trigger, brief duration, Dix-Hallpike torsional upbeating nystagmus, supine-roll geotropic/apogeotropic nystagmus, subjective positional vertigo, atypical positional red flags | Brief triggered positional syndrome, posterior-canal positional pattern, horizontal-canal positional pattern, subjective positional pattern, atypical positional red flag, no objective positional nystagmus | Shows how canal-specific criteria and positional findings can be represented as raw-to-derived-to-diagnosis logic |
| Ménière's disease | Episodic audio-vestibular disorder | Recurrent episodes, 20 min–12 h duration, fluctuating hearing loss, tinnitus, aural fullness, low-frequency SNHL, MRI hydrops, VEMP/ECoG-compatible hydrops finding, migraine features | Episodic vestibular syndrome, auditory cluster, objective low-frequency cochlear pattern, endolymphatic hydrops evidence, migraine overlap pattern | Shows how multi-domain clinical, audiometric, and supportive test evidence can be integrated while avoiding hard exclusion of migraine overlap |
| Presbyvestibulopathy/BVP | Chronic vestibular hypofunction with threshold-based competing diagnosis | Age, chronic dizziness >3 months, gait difficulty/imbalance/falls, oscillopsia or worsening in darkness/uneven ground, mild bilateral vestibular hypofunction, severe bilateral vestibular hypofunction, central neurologic signs | Chronic vestibular course, gait/balance burden, mild bilateral hypofunction, severe bilateral hypofunction, non-central chronic vestibular pattern | Shows how mild versus severe objective vestibular thresholds can separate PVP from BVP as competing diagnostic targets |
