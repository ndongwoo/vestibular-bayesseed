# Python API reference

This document shows common Python usage patterns. It is not a full autogenerated API reference.

## Importing the package

```python
from bayesseed import load_modules, infer_case
```

## Load default modules

```python
modules = load_modules("default_modules")
```

## Run inference for one case

```python
from bayesseed import infer_case, load_modules

modules = load_modules("default_modules")

case = {
    "positional_trigger": 1,
    "brief_duration_seconds_minutes": 1,
    "dix_hallpike_torsional_upbeating_nystagmus": 1,
}

result = infer_case(case, modules)
print(result)
```

## Run inference for one module

```python
from bayesseed.inference import infer_module
from bayesseed.module_loader import load_module

module = load_module("default_modules/bppv.json")

case = {
    "positional_trigger": 1,
    "brief_duration_seconds_minutes": 1,
    "dix_hallpike_torsional_upbeating_nystagmus": 1,
}

result = infer_module(module, case)
print(result)
```

## Batch inference

```python
from bayesseed.inference import batch_infer

results = batch_infer(
    module_path="default_modules",
    case_file="examples/synthetic_cases.csv",
)
```

## Logistic CPD utilities

```python
from bayesseed.logistic_cpd import sigmoid, logistic_probability

p = sigmoid(0.0)
print(p)  # 0.5

p_node = logistic_probability(
    intercept=-2.0,
    weights={"x1": 1.2, "x2": 0.8},
    values={"x1": 1, "x2": 0},
)
```

## Validate a module programmatically

```python
from bayesseed.module_loader import load_module
from bayesseed.schema_validation import validate_module

module = load_module("default_modules/bppv.json", validate=False)
validate_module(module)
```

## Export Mermaid code

```python
from bayesseed.module_loader import load_module
from bayesseed.visualization import to_mermaid

module = load_module("default_modules/bppv.json")
print(to_mermaid(module))
```

## One-way sensitivity analysis

```python
from bayesseed.module_loader import load_modules
from bayesseed.sensitivity import one_way_sensitivity

modules = load_modules("default_modules")

case = {
    "recurrent_spontaneous_episodes": 1,
    "duration_20min_to_12h": 1,
    "fluctuating_hearing_loss": 1,
    "low_frequency_snhl": 1,
    "migraine_features": 1,
}

result = one_way_sensitivity(
    modules=modules,
    case=case,
    edge_id="md_e15",
    beta_values=[0.0, -0.2, -0.5, -0.8],
)

print(result)
```

## Expected result structure

Inference results generally contain:

- target disease probabilities;
- derived-node values;
- edge contributions;
- module metadata.

The exact structure may evolve before v1.0. For stable downstream use, prefer exported CSV or JSON files generated by the CLI or `examples/run_examples.py`.
